# GEOMETRY DASH 3D

Ce projet est une adaptation en 3D du concept de jeu de rythme et de plateforme Geometry Dash. Le but est de guider un cube a travers un niveau rempli d'obstacles, de sauts et de changements de vitesse jusqu'Ã  la zone d'arrivee.

## IMPORTANT : BRANCHE PRINCIPALE

La version fonctionnelle et stable du projet se trouve sur la branche "sansaleatoire".
Veuillez vous assurer de basculer sur cette branche avant de lancer le projet.

Commande git :
git checkout sansaleatoire

## FONCTIONNALITES

Mecaniques de Jeu :
- Systeme de mouvement : Deplacement sur 3 voies avec controles fluides et gestion de la physique (saut, rotation).
- Objets interactifs :
  - Jump Pads : Propulsent automatiquement le joueur en l'air au contact.
  - Orbes de saut : Mecanique de saut flottant avec effets visuels.
  - Portails : Systeme de teleportation (Entree/Sortie).
  - Bonus de vitesse : Zones d'acceleration (boost), de ralentissement et vitesse neutre.

- Obstacles :
  - Piques de differentes tailles et variantes.
  - Plateformes mouvantes avec zones de collision mortelles.

Interface et Systeme :
- Menus complets : Menu principal, selection de niveaux, menu de pause et ecran de fin de niveau.
- Personnalisation : Systeme de skins avec previsualisation 3D en temps reel et sauvegarde des preferences.
- Score : Compteur de morts.
- Audio : Gestion globale de la musique et des effets sonores (menus et niveaux).

Graphismes :
- Environnement 3D avec sol translucide et arriere-plan dynamique.
- Camera suiveuse adaptee aux mouvements du joueur.
- Effets de particules synchronises a l'arrivee.

## INSTALLATION ET LANCEMENT

1. Clonez ce depot.
2. Basculez sur la branche "sansaleatoire".
3. Ouvrez le projet avec Unity.
4. Lancez la scene principale via le menu.

## CONTROLES

- Deplacement lateral : Changer de voie (Touches directionnelles).
- Action : Sauter / Utiliser les orbes (Espace).
- Menu : Touche Echap pour mettre en pause.
